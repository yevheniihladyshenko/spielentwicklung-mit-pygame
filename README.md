# Spielentwicklung mit Python (Pygame)

Arcade-Spiel (Snake-Style) als Portfolio-Projekt. Fokus: saubere Projektstruktur, Menü, Highscore (JSON) und Difficulty-System.

## Features
- Startmenü + Pause + Game Over Screen
- Difficulty: Easy/Normal/Hard
- Highscore Speicherung (JSON)
